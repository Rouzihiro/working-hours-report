# Working Hours Documentation System

A self-contained Typst template system for professional working hours tracking.

## Features

- ✅ **No external dependencies** - 100% self-contained
- ✅ **Professional tables** with checkboxes and alternating colors
- ✅ **CSV import** for data
- ✅ **Automatic numbering** and totals
- ✅ **Clean German formatting** for medical documentation
- ✅ **Easy customization** - change colors, fonts, layouts

## Quick Start

```bash
# Clone or copy this directory
cd working-hours

# Test the system
make test

# Generate your monthly report
make monthly

# Watch for changes during development
make watch-monthly
